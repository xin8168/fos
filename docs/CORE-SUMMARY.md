## 项目愿景（必须保留）

**最高目标**: 零风险、零兜底、零混乱地控制任何设备

**产品定位**: FOS是一个确定性执行主板（非AI推理引擎）
- 纯规则：所有执行逻辑都是硬编码规则
- 无模型：不包含任何机器学习模型
- 零推理：不做任何推理或生成
- 只执行：只执行「明确、成功过」的任务
