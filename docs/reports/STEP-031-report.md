## 3. 调度器功能

### 3.1 核心组件

- `TaskScheduler` - 任务调度器
- `SchedulerConfig` - 调度器配置
- `SchedulerStats` - 统计信息

### 3.2 主要功能

| 功能 | 描述 |
|-----|------|
| start/stop | 启动/停止调度器 |
| submit | 提交普通任务 |
| submit_high_priority | 提交高优先级任务 |
| submit_critical | 提交紧急任务 |
| get_status | 获取任务状态 |
| cancel | 取消任务 |
| stats | 获取统计信息 |
| schedule_loop | 调度循环 |
